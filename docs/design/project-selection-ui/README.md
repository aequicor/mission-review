# Project Selection UI Mockups

These raster files are ImageGen design references for the Compose implementation.
They are not runtime UI backgrounds.

- `project-selection-start-lapis-mockup.png` - start screen reference.
- `project-review-lapis-mockup.png` - local review screen reference.
- `mission-review-icon.png` - generated app icon reference; the desktop app also uses a resized copy as a window icon.

Design direction:

- primary tone: lapis lazuli blue `#26619C`;
- dense productivity layout, not a landing page;
- changed files stay visible independently from the project tree;
- new files are marked with a distinct `New` badge;
- no third-party logos or trademarks.
